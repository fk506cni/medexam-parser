# 指示

以下の「入力テキスト」を、問題ごとに分割してください。
各問題は、多くの場合、行頭に問題番号（数字）のみが記載される形で始まります。

# 出力形式

必ず、以下の仕様に従ったJSON配列として出力してください。

- 全体はJSON配列 `[...]` とします。
- 配列の各要素は、1つの問題を表すJSONオブジェクト `{{...}}` です。
- 各オブジェクトは、以下のキーを持つ必要があります。
  - `problem_number`: 問題番号 (整数)
  - `text`: 問題番号の行から、次の問題の直前までの全文 (文字列)

# 例

## 入力例:
```
これはページのヘッダー情報です。
DKIX-01-AH-7

 1 無症状で発見されることが多い疾患はどれか。
ａ　大動脈解離
ｂ　腹部大動脈瘤
ｃ　感染性心内膜炎

 2 高齢者の入浴中の事故の予防で正しいのはどれか。
ａ　食後に入浴する。
ｂ　深夜に入浴する。

 50 An 85-year-old woman was admitted to the hospital.
Select the most likely cause.
ａ　Cerebral infarction
ｂ　Lumbar disc herniation
```

## 出力例:
```json
[
  {{
    "problem_number": 1,
    "text": "1 無症状で発見されることが多い疾患はどれか。
ａ　大動脈解離
ｂ　腹部大動脈瘤
ｃ　感染性心内膜炎"
  }},
  {{
    "problem_number": 2,
    "text": "2 高齢者の入浴中の事故の予防で正しいのはどれか。
ａ　食後に入浴する。
ｂ　深夜に入浴する。"
  }},
  {{
    "problem_number": 50,
    "text": "50 An 85-year-old woman was admitted to the hospital.
Select the most likely cause.
ａ　Cerebral infarction
ｂ　Lumbar disc herniation"
  }}
]
```

# 重要事項

- 「注意事項」やページ番号、管理番号（例: `DKIX-01-AH-7`）のような、問題文と無関係なヘッダーやフッターは完全に無視してください。
- 入力テキストに問題が一つも見つからない場合（例: 正答値表）は、空の配列 `[]` を返してください。
- 問題番号は必ず行頭から始まります。文中の数字は問題番号ではありません。
- 出力は必ずJSON形式の文字列のみとし、前後に説明文などをつけないでください。
